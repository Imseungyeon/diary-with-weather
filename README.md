# Diary with Weather 
간단한 CRUD API 프로젝트

### 소개
- 해당 날짜의 날씨와 함께 일기 텍스트를 저장/조회/수정/삭제할 수 있는 API
- 외부 API 이용 (https://openweathermap.org/api)

### 개발 기간
- 2024.05.28 ~ 2024.05.30

### 개발 환경
- Java 17
- Spring Boot 3.2.4
- ORM: Jpa
- Database: MySQL
- IDE: IntelliJ
